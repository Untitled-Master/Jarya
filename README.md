# Jarya

<img src="https://i.pinimg.com/736x/42/bb/48/42bb48902adc21603bb7de398eadb4a3.jpg" alt="Jarya" width="100%" />

GPX route viewer with 3D chase-cam simulation, elevation profile, and TikTok-style vertical video export.

## Commands

| Command | Action |
|---|---|
| `npm run dev` | Start dev server (port 5173) |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |
| `npx convex dev` | Start Convex dev server |

## Tech

React 19, Vite 6, Tailwind CSS 3, shadcn/ui (New York, JSX), Leaflet, Convex (TypeScript).
